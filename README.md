# AI Automation Landing Page

Landing page for the AI Automation service under FinanceBeef.com — Services → AI Automation.

Built with **React 18**, **TypeScript**, and **Tailwind CSS 3** via **Vite**.

## Quick Start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Embed in FinanceBeef Main Site

Import the `<AIAutomationPage />` component from `src/components/AIAutomationPage.tsx` and render it under the `/services/ai-automation` route of your main WordPress or React site.
